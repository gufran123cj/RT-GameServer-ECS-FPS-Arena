# nlohmann/json Kütüphanesi

Bu klasöre nlohmann/json kütüphanesini ekleyin:

1. https://github.com/nlohmann/json/releases adresine gidin
2. En son release'i indirin
3. `single_include/nlohmann/json.hpp` dosyasını bu klasöre `json.hpp` olarak kopyalayın

VEYA

Direkt indirme linki (v3.11.3):
https://github.com/nlohmann/json/releases/download/v3.11.3/json.hpp

Bu dosyayı `include/json/json.hpp` olarak kaydedin.
